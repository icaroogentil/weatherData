# Weather Data

Passing the zip code, returns the temperature and humidity

## API Reference

#### get Weather

```http
  GET /weatherData
```

| Parameter | Type     | Description  |
| :-------- | :------- | :----------- |
| `zipCode` | `number` | **Required** |

## Installation

### Install Weather Data with npm

- install dependences

```bash
  npm install
```

- turn on server

```bash
  nodemon app.js
```
